取込 EV分析ツールのエクスポートJSON / 収支グラフJSON

「The競馬 EV分析」の 💾状態を保存 または 収支グラフJSON を取込みます。任意フォルダからファイルを選択(複数可)、または下欄に貼り付け。race_id を主キーに races / entries / results / payouts / bets へ正規化して格納します。

取込 netkeiba 結果ページ HTML

結果ページ(result.html / db.netkeiba)を取込みます。任意フォルダから保存済みHTMLを選択(複数可)、または下欄にソースを貼り付け。着順・払戻・出走馬を抽出して格納します。race_id が本文から取れない場合のみ下欄で補います。

実リンクはブラウザのCORS制限で直取得できない場合があります。その場合は自動で公開CORSプロキシを試し、失敗時は「HTMLを貼り付け」へ誘導します。
